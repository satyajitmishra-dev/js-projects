# Privacy Policy for Launchpad UI

**Last Updated:** December 8, 2024

## Introduction

Launchpad UI ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our Chrome extension.

## Information We Do NOT Collect

Launchpad UI does NOT:
- Collect any personal information
- Track your browsing history
- Store data on external servers
- Use analytics or tracking tools
- Share data with third parties
- Sell your information

## Information Stored Locally

The following data is stored ONLY on your device using Chrome's local storage:

### Tasks/Reminders
- Task descriptions you create
- Task completion status
- Stored locally in your browser
- Never transmitted anywhere

### Custom Quick Apps
- App names and URLs you add
- Stored locally in your browser
- Never transmitted anywhere

### User Preferences
- Dark mode setting
- Card visibility preferences
- Card order (if rearranged)
- Stored locally in your browser
- Never transmitted anywhere

**Important:** All this data stays on YOUR device. We cannot access it.

## Location Data

### Weather Feature
- Uses your device's geolocation to fetch weather data
- Location is sent ONLY to OpenWeatherMap API
- We do NOT store or access your location
- You can deny location permission (weather feature won't work)
- Location is never used for tracking

### How to Control
- Chrome will ask for location permission
- You can allow or deny
- Change anytime in Chrome settings: `chrome://settings/content/location`

## Third-Party Services

Launchpad UI uses these external APIs to provide features:

### OpenWeatherMap API
- **Purpose:** Fetch weather data
- **Data Sent:** Your approximate location (latitude/longitude)
- **Privacy Policy:** https://openweathermap.org/privacy-policy

### Quotable API
- **Purpose:** Fetch daily inspirational quotes
- **Data Sent:** None (public API)
- **Privacy Policy:** https://github.com/lukePeavey/quotable

### GNews API
- **Purpose:** Fetch technology news headlines
- **Data Sent:** None (public API)
- **Privacy Policy:** https://gnews.io/privacy-policy

### Google Favicon API
- **Purpose:** Fetch website icons for Quick Apps
- **Data Sent:** Website URLs you add
- **Privacy Policy:** https://policies.google.com/privacy

**Note:** These services have their own privacy policies. We recommend reviewing them.

## Permissions Explained

### Storage Permission
- **Why:** Save your tasks, apps, and preferences locally
- **Scope:** Local storage only (on your device)
- **Access:** Only this extension can access this data

### Geolocation Permission
- **Why:** Provide accurate weather for your location
- **Scope:** Approximate location only
- **Usage:** Sent only to OpenWeatherMap API
- **Optional:** You can deny this permission

### Host Permissions
- **Why:** Fetch data from weather, quote, and news APIs
- **Scope:** Limited to specific API endpoints
- **Usage:** Only for features you use

## Data Security

- All local data is stored using Chrome's secure storage API
- No data transmission to our servers (we don't have any!)
- No encryption needed (data never leaves your device)
- Data is deleted if you uninstall the extension

## Children's Privacy

Launchpad UI does not knowingly collect information from children under 13. The extension is suitable for all ages.

## Your Rights

You have the right to:
- **Access:** View all data (it's in Chrome's local storage)
- **Delete:** Uninstall the extension to remove all data
- **Control:** Manage permissions in Chrome settings
- **Deny:** Refuse location permission

## Data Deletion

To delete all your data:

**Option 1: Clear Extension Data**
1. Right-click the extension icon
2. Go to "Manage extension"
3. Click "Remove"
4. All local data is deleted

**Option 2: Clear Browser Data**
1. Go to `chrome://settings/clearBrowserData`
2. Select "Cookies and other site data"
3. Clear data
